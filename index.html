神戸観光AIサポート

FlaskとOpenAI APIを使用した、 神戸観光向けのAIチャットサポートアプリです。

制作背景

神戸を訪れる外国人観光客が必要な観光情報を 迅速に取得できるよう支援することを目的として開発しました。 AIチャットを活用することで、観光スポットやアクセス情報を 自然な対話形式で案内できる仕組みを実現しています。

工夫した点

利用者が直感的に操作できるよう、 シンプルで分かりやすいチャット画面を設計しました。 また、メッセージ送信後に自動スクロールを行うことで、 会話の流れをスムーズに確認できるよう工夫しています。 さらに、OpenAI APIを活用し、 自然な対話形式で観光情報を提供できる仕組みを実装しました。

使用技術：
HTML / CSS / JavaScript / Python / Flask / OpenAI API

主な機能

  • AIチャット機能
  • 観光情報案内
  • チャット履歴表示
  • メッセージ送信機能
  • 自動スクロール機能

神戸観光ダッシュボード

神戸市内の観光スポットデータを収集・分析し、 可視化するダッシュボードシステムです。

制作背景

神戸市には多くの観光スポットがありますが、 エリアごとの特徴や人気スポットを比較できるデータを 分かりやすく可視化したツールは多くありません。 そこで本作品では、神戸市内の観光スポット情報を収集し、 Pandasによるデータ分析とFlask・Chart.jsを活用して、 観光スポットの人気度やエリア特性を分析できる ダッシュボードを開発しました。

工夫した点

評価点数だけでなくレビュー件数も考慮した 独自指標「人気指数（Popularity Score）」を設計しました。 また、エリアフィルター機能を実装し、 KPI・カテゴリ分析・人気スポットランキング・ランキング表が 動的に更新される仕組みを構築しました。 Flask APIとJavaScriptを連携させることで、 ページを再読み込みせずにデータを更新できるよう工夫しています。

使用技術：
Python / Flask / Pandas / NumPy
JavaScript / Chart.js / HTML / CSS

主な機能

  • KPI表示
  • カテゴリ別スポット分析
  • エリア別スポット分析
  • 人気スポットTOP10表示
  • 人気観光スポットランキング表示
  • エリアフィルターによる動的分析

神戸観光レポート自動生成ツール

神戸市の観光データを集計・分析し、 ExcelおよびPDF形式のレポートを自動生成するツールです。

制作背景

観光データの集計やレポート作成には多くの時間がかかるため、 作業の効率化を目的として開発しました。 データ分析からレポート出力までを自動化することで、 担当者の作業負担を軽減し、より迅速な意思決定を支援することを目指しています。

工夫した点

Pandasを使用して観光データを集計し、 KPIを自動計算できるように設計しました。 また、Excel形式とPDF形式の両方で出力できるようにすることで、 社内共有や報告資料として活用しやすい構成にしています。 手作業による集計ミスを減らし、 業務効率化につながるツールを目指しました。

使用技術：
Python / Pandas / OpenPyXL / ReportLab

主な機能

  • 観光データ集計
  • KPI自動計算
  • レポート自動生成
  • Excel出力
  • PDF出力
