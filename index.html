神戸観光AIサポート

FlaskとOpenAI APIを使用した、 神戸観光向けのAIチャットサポートアプリです。

制作背景

工夫した点

利用者が直感的に操作できるよう、 シンプルで分かりやすいチャット画面を設計しました。 また、メッセージ送信後に自動スクロールを行うことで、 会話の流れをスムーズに確認できるよう工夫しています。 さらに、OpenAI APIを活用し、 自然な対話形式で観光情報を提供できる仕組みを実装しました。

神戸を訪れる外国人観光客が必要な観光情報を 迅速に取得できるよう支援することを目的として開発しました。 AIチャットを活用することで、観光スポットやアクセス情報を 自然な対話形式で案内できる仕組みを実現しています。

使用技術： HTML / CSS / JavaScript / Python / Flask / OpenAI API

主な機能

  • AIチャット機能
  • 観光情報案内
  • チャット履歴表示
  • メッセージ送信機能
  • 自動スクロール機能

神戸観光Dashboard

神戸市の観光データを分析し、 KPIやグラフを可視化するダッシュボードです。

制作背景

神戸市の観光データを可視化し、 観光動向を分かりやすく分析できる環境を提供することを目的として開発しました。 ダッシュボード形式で情報を表示することで、 利用者が観光客数や地域別傾向を直感的に把握できるようにしています。

工夫した点

利用者が重要な情報を素早く把握できるよう、 KPIを画面上部に配置しました。 また、Chart.jsを活用してグラフを可視化し、 月別推移や地域別データを直感的に分析できるよう工夫しています。 フィルター機能を実装することで、 必要なデータのみを効率的に確認できる構成としました。

使用技術： Python / Pandas / Chart.js

主な機能

  • 観光客数の集計
  • 月別推移分析
  • 地域別分析
  • KPI表示
  • グラフ可視化

神戸観光レポート自動生成ツール

神戸市の観光データを集計・分析し、 ExcelおよびPDF形式のレポートを自動生成するツールです。

制作背景

観光データの集計やレポート作成には多くの時間がかかるため、 作業の効率化を目的として開発しました。 データ分析からレポート出力までを自動化することで、 担当者の作業負担を軽減し、より迅速な意思決定を支援することを目指しています。

工夫した点

Pandasを使用して観光データを集計し、 KPIを自動計算できるように設計しました。 また、Excel形式とPDF形式の両方で出力できるようにすることで、 社内共有や報告資料として活用しやすい構成にしています。 手作業による集計ミスを減らし、 業務効率化につながるツールを目指しました。

使用技術： Python / Pandas / OpenPyXL / ReportLab

主な機能

  • 観光データ集計
  • KPI自動計算
  • レポート自動生成
  • Excel出力
  • PDF出力
